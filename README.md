# React-Terminal
Component is almost ready to be published to npm. Until then, to run and test the project locally:
```bash
git clone https://github.com/zaidsidd360/react-terminal.git
```
```bash
cd react-terminal
```
```bash
npm i
```
or
```bash
yarn
```
then
```bash
cd example
```
repeat `npm i` or `yarn`
and finally,
```bash
npm run dev
```
or 
```bash
yarn dev
```
